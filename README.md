# cpr

A fast replacement for cp -R.

## Description

Cpr is a drop in replacement for the unix utility cp with the recursive (-R) option (ie. cp -R). The difference? It's about five to six times faster.

### Example
```
$ cpr ~/big-dir ~/big-dir-copy
```
